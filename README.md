Este proyecto es un portafolio web personal, moderno y responsivo, diseñado para mostrar mi trayectoria y proyectos como desarrollador.

## Tecnologías Utilizadas

Este proyecto ha sido construido utilizando estándares modernos de desarrollo web y las últimas versiones de herramientas clave:

- **React 19**: Biblioteca principal para la construcción de interfaces de usuario.
- **Vite 6**: Entorno de desarrollo de última generación para una experiencia rápida y optimizada.
- **TypeScript 5**: Superset de JavaScript que añade tipado estático robusto.
- **Tailwind CSS 3**: Framework de estilos 'utility-first' para un diseño ágil y responsivo.
- **i18next**: Implementación de internacionalización (Soporte Español/Inglés).
